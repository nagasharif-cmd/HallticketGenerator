# Hall Ticket Generator

Android app for generating student hall tickets.

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- MVVM

## Features
- Add Students
- Store Exam Details
- Generate Hall Tickets
- Offline Data Storage
